## Add Bank Account

<RequestExample>
```bash cURL
curl -X POST https://api.sociapp.com/configuracion/bancos \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "Nombre": "Banco Santander",
    "Direccion": "Gran Vía 25",
    "CodigoPostal": 28013,
    "Poblacion": "Madrid",
    "Pais": "España",
    "Telefono": "+34 915551234",
    "CodigoNegocio": "0049",
    "Referencia_SEPA": "ES12345678",
    "IBAN": "ES7921000813610123456789",
    "Swift": "BSCHESMM"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sociapp.com/configuracion/bancos', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    Nombre: 'Banco Santander',
    Direccion: 'Gran Vía 25',
    CodigoPostal: 28013,
    Poblacion: 'Madrid',
    Pais: 'España',
    Telefono: '+34 915551234',

    Referencia_SEPA: 'ES12345678',
    IBAN: 'ES7921000813610123456789',

```
</RequestExample>

## Delete Bank Account

<RequestExample>
```bash cURL
curl -X DELETE https://api.sociapp.com/configuracion/bancos/2 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.sociapp.com/configuracion/bancos/2', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "message": "Bank account deleted successfully"
}
```
</ResponseExample>

### Endpoint

```http
DELETE /configuracion/bancos/:id
```

### Authentication

Requires JWT authentication token.

### Path Parameters

<ParamField path="id" type="number" required>
  Bank account record ID to delete
</ParamField>
