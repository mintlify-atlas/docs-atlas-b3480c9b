## Add Donation

<RequestExample>
```bash cURL
curl -X POST https://api.sociapp.com/configuracion/donativos \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "Nombre": "Fundación Solidaria",
    "Direccion": "Plaza España 10",
    "Poblacion": "Sevilla",
    "CP": 41001,
    "Pais": "España",
    "Telefono": "+34 954123456",
    "NIF": "G87654321",
    "Email": "info@fundacionsolidaria.org",
    "Notas": "Colaboración para proyecto educativo",
    "Tipo": "Donativo"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.sociapp.com/configuracion/donativos', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    Nombre: 'Fundación Solidaria',
    Direccion: 'Plaza España 10',
    Poblacion: 'Sevilla',
    CP: 41001,
    Pais: 'España',
    Telefono: '+34 954123456',
    NIF: 'G87654321',
    Email: 'info@fundacionsolidaria.org',
    Notas: 'Colaboración para proyecto educativo',
    Tipo: 'Donativo'
  })
});
const data = await response.json();
```
</RequestExample>

<ResponseExample>

  Contact phone number
</ParamField>

<ParamField body="NIF" type="string">
  Tax identification number (NIF/CIF)
</ParamField>

<ParamField body="Email" type="string">
  Contact email address (must be valid email format)
</ParamField>

</ParamField>

## Delete Donation

<RequestExample>
```bash cURL
curl -X DELETE https://api.sociapp.com/configuracion/donativos/3 \
  -H "Authorization: Bearer YOUR_TOKEN"
```

```javascript JavaScript
const response = await fetch('https://api.sociapp.com/configuracion/donativos/3', {
  method: 'DELETE',
  headers: {
    'Authorization': 'Bearer YOUR_TOKEN'
  }
});
```
</RequestExample>

<ResponseExample>
```json 200 OK
{
  "message": "Donation record deleted successfully"
}
```
</ResponseExample>

### Endpoint

```http
DELETE /configuracion/donativos/:id
```

### Authentication

Requires JWT authentication token.

### Path Parameters

<ParamField path="id" type="number" required>
  Donation record ID to delete
</ParamField>

## Donation Types

The `Tipo` field accepts the following enum values:

- **Donativo**: Regular donation from an individual or organization
- **Herencia**: Inheritance or bequest
