---
title: 'Create Activity'
api: 'POST /activities'
---

Creates a new activity and optionally associates it with projects.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

**Required Roles:** `monitor`, `admin`

## Request Body

<ParamField body="name" type="string" required>
  Name of the activity
</ParamField>

<ParamField body="place" type="string" required>
  Location where the activity takes place
</ParamField>

<ParamField body="horaInicio" type="string" required>
  Start time in HH:MM or HH:MM:SS format. Must be before horaFin.
</ParamField>

<ParamField body="horaFin" type="string" required>
  End time in HH:MM or HH:MM:SS format. Must be after horaInicio.
</ParamField>

<ParamField body="diaSemana" type="string" required>
  Day of the week when the activity occurs
</ParamField>

<ParamField body="idMonitor" type="number">
  ID of the monitor assigned to this activity
</ParamField>

<ParamField body="projectIds" type="number[]">
  Array of project IDs to associate with this activity
</ParamField>

<ParamField body="icon" type="string">
  Icon identifier for the activity
</ParamField>

## Response

<ResponseField name="id" type="number">
  Unique identifier for the created activity
</ResponseField>

<ResponseField name="name" type="string">
  Name of the activity
</ResponseField>

<ResponseField name="place" type="string">
  Location where the activity takes place
</ResponseField>

<ResponseField name="horaInicio" type="string">
  Start time in HH:MM format
</ResponseField>

<ResponseField name="horaFin" type="string">
  End time in HH:MM format
</ResponseField>

<ResponseField name="diaSemana" type="string">
  Day of the week
</ResponseField>

<ResponseField name="icon" type="string">
  Icon identifier for the activity
</ResponseField>

<ResponseField name="idMonitor" type="number">
  ID of the assigned monitor
</ResponseField>

<ResponseField name="monitor" type="string">
  Name of the assigned monitor
</ResponseField>

<ResponseField name="proyectos" type="array">
  Array of associated projects
  
  <ResponseField name="id" type="number">
    Project ID
  </ResponseField>
  
  <ResponseField name="nombre" type="string">
    Project name
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X POST https://api.sociapp.com/activities \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "Fútbol",
    "place": "Campo deportivo",
    "horaInicio": "16:00",
    "horaFin": "18:00",
    "diaSemana": "Lunes",
    "icon": "soccer",
    "idMonitor": 5,
    "projectIds": [1, 3]
  }'
```

## Example Response

  "horaFin": "18:00",

## Validation Rules

- All required fields (`name`, `place`, `horaInicio`, `horaFin`, `diaSemana`) must be provided
- `horaInicio` must be before `horaFin`
- Time format must be HH:MM or HH:MM:SS with valid hours (00-23) and minutes (00-59)
