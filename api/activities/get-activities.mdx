---
title: 'Get Activities'
api: 'GET /activities'
---

Retrieves all activities with their associated monitors and projects.

## Authentication

<ParamField header="Authorization" type="string" required>
  Bearer token for authentication
</ParamField>

**Required Roles:** `monitor`, `admin`

## Response

<ResponseField name="activities" type="array" required>
  Array of activity objects
  
  <ResponseField name="id" type="number">
    Unique identifier for the activity
  </ResponseField>
  
  <ResponseField name="name" type="string">
    Name of the activity
  </ResponseField>
  
  <ResponseField name="place" type="string">
    Location where the activity takes place
  </ResponseField>
  
  <ResponseField name="horaInicio" type="string">
    Start time in HH:MM format
  </ResponseField>
  
  <ResponseField name="horaFin" type="string">
    End time in HH:MM format
  </ResponseField>
  
  <ResponseField name="diaSemana" type="string">
    Day of the week
  </ResponseField>
  
  <ResponseField name="icon" type="string">
    Icon identifier for the activity
  </ResponseField>
  
  <ResponseField name="idMonitor" type="number">
    ID of the assigned monitor
  </ResponseField>
  
  <ResponseField name="monitor" type="string">
    Name of the assigned monitor
  </ResponseField>
  
  <ResponseField name="proyectos" type="array">
    Array of associated projects
    
    <ResponseField name="id" type="number">
      Project ID
    </ResponseField>
    
    <ResponseField name="nombre" type="string">
      Project name
    </ResponseField>
  </ResponseField>
</ResponseField>

## Example Request

```bash
curl -X GET https://api.sociapp.com/activities \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```
